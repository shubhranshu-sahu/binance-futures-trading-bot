# Binance Futures Trading Bot (CLI)

A structured Python CLI trading bot built for Binance Futures Testnet (USDT-M) using direct REST API integration with HMAC-SHA256 request signing.

## Features

- Place MARKET and LIMIT orders
- Supports BUY and SELL sides
- Direct REST API integration (no SDK abstraction)
- HMAC-SHA256 request signing
- Structured project architecture
- Input validation
- Centralized logging to file
- Robust error handling

## Tech Stack

- Python 3.x
- requests
- python-dotenv
- Binance Futures REST API

## Environment

Base URL (Demo Futures): https://demo-fapi.binance.com
